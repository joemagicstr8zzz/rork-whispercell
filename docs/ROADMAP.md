# Speed Reader 3D Roadmap

## Done in the first real build

- React/Vite app foundation
- 3D focus tunnel reading view
- WPM control
- chunk-size control
- punctuation-aware pauses
- progress bar
- remaining time estimate
- total session estimate
- keyboard controls
- local saved settings/text
- responsive phone-friendly layout
- app icon and web manifest
- GitHub Actions build check

## Next high-impact upgrades

### 1. Library mode
Save multiple reading pieces instead of only one pasted text.

### 2. Guided training mode
A session that starts slow, gently increases WPM, then cools down.

### 3. Focus word highlighting
Highlight the ideal eye-fixation letter inside each word or chunk.

### 4. Import helpers
Import from pasted URLs, PDFs, clipboard text, or local text files.

### 5. Reading analytics
Track sessions, average WPM, completion streaks, and comfort scores.

### 6. Mobile install polish
Add offline support and a better install prompt so it feels app-like on phones.

### 7. AI assist mode
Optional summary, difficult-word explanations, and quiz generation after a reading session.
